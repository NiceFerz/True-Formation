# TruthCheck: Сервис проверки достоверности информации

Учебный проект в рамках проектной практики первого курса по направлению «Информационные системы и технологии»[cite: 1].

## О проекте
TruthCheck — это доступный инструмент, который помогает пользователям критически оценивать контент, который они видят в интернете. Сервис объединяет несколько методов проверки: анализ источника, обнаружение AI-генерации и глубокий поиск артефактов.
