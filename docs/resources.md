# Ресурсы

Полезные материалы по теме проекта, инструментам разработки и технологиям, использованным в ходе проектной практики.

---

## По теме проекта

### Верификация информации и deepfake

- **[Snopes](https://snopes.com)** — один из старейших и авторитетных fact-checking ресурсов. Пример того, как устроена верификация информации на практике.

- **[Медиаграмотность.рф](https://www.media-literacy.ru)** — российский ресурс о медиаграмотности, распознавании фейков и критическом мышлении.

- **[FakeCheck](https://fakecheck.ru)** — российский сервис проверки фейков, аналог нашего проекта. Полезен для понимания рынка и существующих решений.

- **[Deepware Scanner](https://deepware.ai)** — инструмент для обнаружения deepfake-видео. Показывает технический подход к решению проблемы поддельного контента.

### Видео

- **[Что такое deepfake и как его распознать](https://www.youtube.com/watch?v=cQ54GDm1eL0)** — видео объясняет технологию deepfake и методы обнаружения поддельного контента. Ключевая тема нашего проекта.

## Инструменты практики

### Git и GitHub

- **[Официальная документация Git](https://git-scm.com/book/ru/v2)** — полная книга по Git на русском языке от создателей системы контроля версий.

- **[Бесплатный курс по Git на Hexlet](https://ru.hexlet.io/courses/intro_to_git)** — интерактивный курс по основам Git для начинающих разработчиков.

- **[Что такое Git — объяснение на схемах](https://skillbox.ru/media/code/chto_takoe_git_obyasnyaem_na_skhemakh/)** — статья с визуальным объяснением принципов работы Git.

### HTML и CSS

- **[Элементы HTML — MDN](https://developer.mozilla.org/ru/docs/Web/HTML/Element)** — справочник по всем HTML-элементам от Mozilla. Лучший источник по стандартам веб-разметки.

- **[Основы HTML — MDN](https://developer.mozilla.org/ru/docs/Learn_web_development/Getting_started/Your_first_website/Creating_the_content)** — руководство по созданию первого сайта.

- **[Основы CSS — MDN](https://developer.mozilla.org/ru/docs/Web/CSS)** — полная документация по CSS на русском языке: свойства, селекторы, анимации.

- **[Введение в CSS-вёрстку — MDN](https://developer.mozilla.org/ru/docs/Learn_web_development/Core/CSS_layout/Introduction)** — руководство по основам компоновки страниц.

- **[Doka Guide](https://doka.guide)** — русскоязычная документация по веб-технологиям HTML, CSS, JS. Написана понятно для начинающих.

- **[DevTools для чайников](https://habr.com/ru/articles/548898/)** — статья на Habr об инструментах разработчика в браузере.

### Markdown

- **[Уроки по Markdown — Hexlet](https://ru.hexlet.io/lesson_filters/markdown)** — практические уроки по синтаксису Markdown для написания документации.
