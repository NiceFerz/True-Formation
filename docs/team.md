# Участники проекта

Команда проекта TruthCheck — три студента первого курса направления «Информационные технологии и информационная безопасность».

---

## Войков Никита Григорьевич

**Роль:** Git & Документация (пункт 1)

### Задачи:
- Создание репозитория на GitHub по шаблону
- Настройка структуры папок проекта
- Освоение базовых команд Git: клонирование, коммит, пуш, ветки
- Ведение истории коммитов с осмысленными сообщениями

### Вклад в проект:
Никита обеспечил техническую основу совместной работы команды — благодаря правильно настроенному репозиторию все участники могли вносить изменения параллельно без конфликтов.

---

## Найденов Андрей Дмитриевич

**Роль:** Документация & Финальный отчёт (пункты 2 и 5)

### Задачи:
- Изучение синтаксиса Markdown
- Написание описания проекта в формате Markdown
- Ведение журнала прогресса (`JOURNAL.md`)
- Исследование существующих решений в области верификации информации
- Составление финального отчёта по практике по шаблону DOCX
- Формирование PDF-версии отчёта
- Загрузка отчётов в LMS и репозиторий

### Вклад в проект:
Андрей отвечал за смысловое наполнение проекта — провёл анализ конкурентов, сформулировал цели и задачи, описал архитектуру сервиса. Вся документация проекта оформлена им в читаемом и структурированном виде.

## Ситникова Софья Павловна

**Роль:** Сайт & Дизайн (пункт 3)

### Задачи:
- Изучение инструментов: VS Code, Live Server
- Разработка дизайн-системы сайта (цвета, шрифты, компоненты)
- Вёрстка всех 5 страниц сайта на HTML и CSS:
  - Главная страница с аннотацией проекта
  - Страница «О проекте»
  - Страница «Участники»
  - Страница «Журнал» (4 записи о прогрессе)
  - Страница «Ресурсы»
- Добавление графических материалов: SVG-схемы, диаграммы
- Встраивание видеоматериала по теме проекта

### Вклад в проект:
Софья реализовала визуальную часть проекта — сайт в тёмной теме с зелёными акцентами, вдохновлённый дизайном мобильного приложения TruthCheck. Самый объёмный пункт практики выполнен в полном объёме.
