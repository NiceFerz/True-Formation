# О проекте TruthCheck

## Аннотация

**TruthCheck** — сервис для автоматической проверки достоверности цифрового контента. Учебный проект в рамках проектной практики первого курса по направлению «Информационные технологии и информационная безопасность».

---

## Проблема

Количество фейков, deepfake-видео и AI-сгенерированного контента растёт с каждым годом. Обычным пользователям становится всё сложнее отличить достоверную информацию от манипуляций.

TruthCheck — это попытка создать доступный инструмент, который поможет людям критически оценивать контент, который они видят в интернете.

---

## Решение

Сервис объединяет несколько методов проверки и агрегирует результаты в единый понятный балл — **Trust Score** от 0 до 100.

Пользователю не нужны специальные знания — интерфейс даёт чёткий ответ и объясняет, что именно повлияло на оценку.

---

## Цели и задачи

**Цель:** разработать концепцию и прототип сервиса для автоматической проверки достоверности цифрового контента с понятным интерфейсом.

**Задачи:**

- Изучить существующие методы верификации информации, анализа медиафайлов и обнаружения дезинформации
- Спроектировать архитектуру системы и алгоритм формирования Trust Score
- Разработать пользовательский интерфейс сервиса
- Оформить всю проектную документацию и создать сайт проекта

---

## Возможности сервиса

| Модуль | Описание |
|---|---|
| 📄 Анализ текста | Распознавание признаков AI-генерации, проверка фактов |
| 🔗 Проверка ссылок | Оценка репутации домена, история ресурса |
| 🖼️ Анализ изображений | Обнаружение монтажа, поиск оригинала |
| 🎬 Обнаружение deepfake | Проверка видео на артефакты генерации |
| 📊 Trust Score | Агрегированный балл доверия от 0 до 100 |
| 📁 История проверок | Сохранение результатов с экспортом отчётов |

---

## Как это работает

1. **Загрузка контента** — пользователь вставляет ссылку или загружает файл (текст, фото, видео)
2. **Анализ источника** — проверяем репутацию источника в базах данных
3. **Обнаружение артефактов** — ИИ проверяет признаки генерации и deepfake
4. **Trust Score** — агрегируем результаты в итоговый балл с рекомендациями

---

## Уровни доверия

| Балл | Уровень | Значение |
|---|---|---|
| 80–100 | 🟢 Высокий | Контент скорее всего достоверен |
| 40–79 | 🟡 Средний | Требует дополнительной проверки |
| 0–39 | 🔴 Низкий | Высокий риск дезинформации |
